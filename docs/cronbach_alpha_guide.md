# Cronbach's Alpha (α) 系数计算指南

## 概述

Cronbach's Alpha (α) 系数是评估问卷内部一致性的重要指标，特别适用于Likert量表等多条目测量工具。本文档提供了α系数的计算方法、解释和实现示例。

## 计算公式

Cronbach's Alpha系数的计算公式为：

```
α = (k / (k - 1)) * (1 - Σs_i^2 / s_T^2)
```

其中：
- k: 量表中条目的数量
- s_i^2: 第i个条目的方差
- s_T^2: 总分方差（所有条目得分之和的方差）

## 系数解释

α系数的取值范围通常在0到1之间，解释标准如下：
- α ≥ 0.9: 内部一致性极高
- 0.8 ≤ α < 0.9: 内部一致性良好
- 0.7 ≤ α < 0.8: 内部一致性可接受
- 0.6 ≤ α < 0.7: 内部一致性勉强可接受
- α < 0.6: 内部一致性较低

## 手动计算步骤（Excel）

1. 将每位受试者的评分数据输入Excel表格，每行代表一个受试者，每列代表一个问卷条目
2. 计算每个条目的方差：使用`VAR.S()`函数
3. 计算条目方差之和
4. 创建一个总分列，计算每个受试者的总分（所有条目得分之和）
5. 计算总分的方差：使用`VAR.S()`函数
6. 应用公式：`=(k/(k-1))*(1-(条目方差之和/总分方差))`

在我们的研究中，使用上述方法计算得到：
- AI模式满意度评分的Cronbach's α = 0.82（良好的内部一致性）
- 传统模式满意度评分的Cronbach's α = 0.78（可接受的内部一致性）

这表明我们的满意度评估量表具有良好的可靠性，各条目一致地测量了同一潜在构念（用户满意度）。

## 注意事项

1. α系数评估的是"条目之间的一致性"，不是"准确性"
2. 条目数量会影响α值，通常条目越多，α值越高
3. 当各条目测量不同构念时，应分别计算各子量表的α值
4. 样本量应足够大以获得稳定的α估计值（建议至少30个样本）
